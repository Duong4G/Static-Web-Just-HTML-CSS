Các thành phần của Java Spring



Spring được chia thành nhiều module khác nhau, tùy theo mục đích phát triển ứng dụng mà ta dùng một trong các module đó.

1. Test:

Tầng này cung cấp khả năng hỗ trợ kiểm thử với JUnit và TestNG.

2. Spring core container:

Bao gồm các module spring core, beans, context và expression languate(EL)

  • Spring core, bean cung cấp tính năng IOC và Dependency Injection
  • Spring context hỗ trợ đa ngôn ngữ và các tính năng của Java EE
  • Expresstion Language hỗ trợ việc getting và setting giá trị, các toán tử logic,…

3. AOP, Aspects and Instrumentation:

Những module này hỗ trợ lập trình hướng khía cạnh (Aspect Oriented Programming).

4. Data Access/Integration:

Cung cấp khả năng giao tiếp với Database.

5. Web (Spring MVC):

Hỗ trợ việc tạo ứng dụng web.
