Spring Tool Suite

1. Cách cài đặt Spring Tool Suite:

Spring Tool Suite là một môi trường phát triển dựa trên Eclipse có được tùy biến cho phát triển ứng dụng Spring. Nó cung cấp một môi trường để thực thi và triển khai các ứng dụng Spring.

Để cài đặt Spring Tool Suite ta cần phải có phần mềm Eclipse trong máy.

Các bước cài đặt:

  • Mở IDE Eclipse
  • Chọn Help --> Eclipse Marketplace
  • Trong cửa sổ Eclipse Marketplace, gõ STS trong trường Find --> Nhấn nút Go.
  • Click nút install
  • Trong màn hình tiếp theo, nhấp vào chọn tất cả --> Nhấn nút Confirm
  • Trong màn hình tiếp theo, chọn I accept the terms of the license agreements --> Click Finish
  • Chờ cho phần mềm được cài đặt và Khởi động lại Eclipse.

2. Sử dụng java spring bằng Spring Tool Suite:

Hướng dẫn tạo một Spring Project:

  • File/New/Other… -> Chọn Spring Starter Project -> Next
  • Điền tên project vào cột name -> Next
  • Lựa chọn phiên bản cho phù hợp, tích vào ô Web -> Next
  • Nhấn Finish để kết thúc.
